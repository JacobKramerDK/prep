# Product Overview

## Product Purpose
Prep is a desktop meeting preparation assistant that transforms how knowledge workers approach meetings. By connecting to Obsidian vaults and calendars, it automatically surfaces relevant context from past notes and conversations, then uses AI to generate comprehensive meeting briefs. The tool eliminates the reactive scramble before meetings and ensures users walk in prepared with full context.

## Target Users
Knowledge workers and leaders who manage multiple meetings daily:
- **Managers and Team Leads**: Need to track team discussions, decisions, and action items across multiple projects
- **Executives**: Require quick context switching between strategic discussions with different stakeholders
- **Consultants and Client-facing Professionals**: Must recall detailed client history and project context
- **Anyone using Obsidian**: Users who already maintain extensive note systems but struggle to surface relevant information quickly

**User Needs**: Quick context recall, meeting preparation efficiency, action item tracking, seamless integration with existing note-taking workflows.

## Key Features
- **Obsidian Vault Integration**: Direct connection to existing markdown note systems
- **Calendar Parsing**: Automatic meeting detection and context association
- **AI-Powered Meeting Briefs**: Comprehensive preparation summaries using OpenAI
- **Context Surfacing**: Intelligent retrieval of relevant past notes and conversations
- **Audio Transcription**: Post-meeting summary generation using Whisper
- **Offline Vault Browsing**: Works without internet for note review and calendar viewing
- **Cross-Platform Support**: Native desktop experience on macOS and Windows

## Business Objectives
- **User Productivity**: Reduce meeting preparation time by 70% while increasing preparation quality
- **Knowledge Retention**: Help users leverage their existing note systems more effectively
- **Meeting Effectiveness**: Improve meeting outcomes through better preparation and context awareness
- **Workflow Integration**: Seamlessly fit into existing Obsidian and calendar workflows

## User Journey
1. **Setup**: Connect Prep to Obsidian vault and calendar system
2. **Automatic Detection**: Prep identifies upcoming meetings from calendar
3. **Context Gathering**: App searches vault for relevant notes, past meeting records, and related discussions
4. **AI Brief Generation**: Creates comprehensive meeting preparation document with key context, talking points, and action items
5. **Meeting Execution**: User enters meeting fully prepared with relevant background
6. **Post-Meeting**: Optional audio transcription and summary integration back into vault

## Success Criteria
- **Time Savings**: Users spend 50% less time manually preparing for meetings
- **Context Accuracy**: 90% of surfaced context rated as relevant by users
- **User Adoption**: Daily active usage for users with 5+ meetings per week
- **Performance**: Handle 1000+ file vaults with sub-3-second search times
- **Cross-Platform Reliability**: Consistent experience across macOS and Windows
